# Laser DAC Shield R1 (KiCad Starter)

This folder contains a starter KiCad project:

- `laser_dac_shield_r1.kicad_pro`
- `laser_dac_shield_r1.kicad_sch`

## How to use
1. Open `laser_dac_shield_r1.kicad_pro` in KiCad 8 or newer.
2. Create/complete the PCB in PCB Editor and save as `laser_dac_shield_r1.kicad_pcb`.
3. Run ERC/DRC.
4. Export fabrication files into `fabrication/`.
5. Export assembly files into `assembly/`.

## Note
A `.kicad_pro` alone is not sufficient for manufacturing. You must also provide final Gerbers, drill files, BOM, and pick/place files.
